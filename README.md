# Clima Horário Brasil

**Dados climáticos horários em tempo real.**

Plataforma de dados climáticos horários do Brasil, com informações coletadas a partir
das estações meteorológicas do INMET.

## Funcionalidades
- Dados horários por estação
- Visualização em mapa interativo
- Registro de extremos climáticos
- API REST em Flask

## Tecnologias
- Python
- Flask
- Leaflet
- API INMET

## Fonte dos dados
INMET — Instituto Nacional de Meteorologia
